# mkd-progress
> A mkd component

## Installation
```shell
npm i mkd-progress -D
```

## Usage
```javascript
import Vue from 'vue'
import MintProgress from 'mkd-progress'

Vue.use(MintProgress)
```

or

```javascript
import Vue from 'vue'
import { MintProgress } from 'mkd-progress'

Vue.component('mkd-progress', MintProgress)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
