# mkd-range
> A mkd component

## Installation
```shell
npm i mkd-range -D
```

## Usage
```javascript
import Vue from 'vue'
import MintRange from 'mkd-range'

Vue.use(MintRange)
```

or

```javascript
import Vue from 'vue'
import { MintRange } from 'mkd-range'

Vue.component('mkd-range', MintRange)
```

## Development

```shell
make dev
```

## Production
```
make dist
```

## Deploy
```shell
make deploy
```
